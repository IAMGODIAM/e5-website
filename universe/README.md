# universe/ — The Interconnected Universe suite

Source for **universe.e5enclave.com** (E5's Gilded Editorial light suite: home, McCartney Academy,
Oral-History Archive, Business Plan, Schools of Hope).

Deployed by a **git-connected Cloudflare Pages project** (separate from the brand site):
- root_dir: `universe`  |  build: none (static)  |  output: `public`
- Pushes to `main` auto-deploy this folder to universe.e5enclave.com.

Kept in the e5-website repo (shared) rather than a standalone repo. Design canon: light "Gilded Editorial".
